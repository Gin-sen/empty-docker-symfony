# Dockeriser Symfony
## Prérequis
- Docker et DockerCompose installé
- Composer installé

## Installation (mode dev)
A la racine du projet :
```bash
make install
```

## Lancer le projet
A la racine du projet :
```bash
make
```
et rendez-vous sur `http://localhost:8080`

Le code se met à jour à chaque sauvegarde de fichier car le volume est monté.
